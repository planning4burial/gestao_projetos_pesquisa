## Descrição

O  sistema  proposto  consiste  em  uma  plataforma  institucional  voltada  para  o ambiente  universitário,  cujo  objetivo  principal  é  automatizar,  organizar  e  auditar  a  produção científica  e  as  atividades  cotidianas  de  grupos  de  pesquisa.  A  plataforma  atua  desde  o  nível operacional  —  onde  discentes  realizam  e  atualizam  tarefas  diárias  —  até  o  nível estratégico,  permitindo  que  a  alta  gestão  da  instituição  monitore  indicadores  de produtividade global e a distribuição de recursos/bolsas. Público-alvo: O corpo acadêmico universitário.
